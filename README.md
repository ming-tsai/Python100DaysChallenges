## 📖 About

This repository documents my **#100DaysOfCode** Python journey, following the structure of Dr. Angela Yu's *"100 Days of Code: The Complete Python Pro Bootcamp"*. Each day includes a mini project, exercises, and notes — built from scratch, one day at a time. 💪

---

Each day folder contains:
- `main.py` — the day's project
- `README.md` *(optional)* — notes, what I learned, challenges faced
